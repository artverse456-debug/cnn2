# Creator Platform UI

Dieses Repository enthält das Frontend-Projekt **creator-platform-ui**, ein mit Next.js 14, TypeScript, TailwindCSS und Zustand gebautes Demo-Interface für Creator- & Fan-Dashboards.

## Quickstart

```bash
cd creator-platform-ui
npm install
npm run dev
```

> Hinweis: Alle Daten sind statisch und werden direkt im Frontend simuliert.
